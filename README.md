## Installation

```console
$ npm install redux-csrf --save
```

## Usage

```js
import { default as reducer, actions } from 'redux-csrf';

// Server side with csurf middleware
store.dispatch(actions.setCsrfToken(req.csrfToken()));

// Reducer
combineReducers({
  ...otherReducers
  _csrf: reducer,
});
```

## API

### `setCsrfToken(token)`

Set CSRF token to redux store.
